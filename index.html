Automatisation du dépot de projets de recherche

Mise en place d'une plateforme web responsive permettant aux chercheurs du CHU de réaliser des demandes pour des projets de recherche. S'il n'existe pas, un compte est créé pour l'utilisateur afin de pouvoir stocker en base de données l'ensemble de ses informations et de pouvoir ainsi retrouver ses demandes passées. Pour chaque demande faite, un mail est envoyé automatiquement à la fois au demandeur, mais également à l'organisme chargé de traiter ces demandes.

Production d’un diagramme de Sankey (diagramme de flux) pour des données de santé

Ce projet consistait à générer un diagramme de Sankey (figure ci-dessous) à partir d'un programme Python 3.6 et de données de santé banalisées. Le préalable fut de comprendre le programme existant qui permettait la génération d'un premier fichier Excel contenant les données d'entrée du générateur. Ces données devaient correspondre parfaitement aux paramètres de la fonction de génération. La bibliothèque utilisée génère un fichier HTML qui permet de visualiser les données de santé de manière intuitive.
